Detailed Description

System responsible managing explicit time integration.

Each system holds a mesh and set of external forces and advancing the solution in time by explicitly integrating the equations of motion using the Newmark-β method.

Constructor.

Constructs a system containing the specified mesh and applied external forces. To avoid copy construction of what, in general, are expensive objects both the mesh and set of external forces must be cast as r-values to be "moved" into the system. In effect the explicit system acts as a sink for both input parameters.

Parameters
[in]mesh.Mesh constructed from a set of node and element lists in coordination with a set of boundary conditions. Responsible for providing the global stiffness and mass matrices.
[in]external_forces.Vector of nodal forces applied to the model.
[in]initial_displacementsInitial nodal displacements.
[in]initial_velocitiesInitial nodal velocities.
[in]t0Initial time of the system.
[in]optionsOptions used to customize behavior of the system.

ColumnVector const & explicit_fea::ExplicitSystem::getDisplacements ( ) const

Returns the current nodal displacements.

Nodal displacements are held in a 1D vector of the form
\( \left[ \delta u_{1}^{x}, \delta u_{1}^{y}, \delta u_{1}^{z}, \delta \theta_{1}^{x}, \delta \theta_{1}^{y}, \delta \theta_{1}^{z}, ..., \delta u_{N}^{x}, \delta u_{N}^{y}, \delta u_{N}^{z}, \delta \theta_{N}^{x}, \delta \theta_{N}^{y}, \delta \theta_{N}^{z}, \right] \)
where \( \delta u \) specifies displacement along translational degrees of freedom for the given node number \( \left( 1, 2, 3, ...N\right) \) and direction \( \left( x, y, z) \) and \( \delta \theta \) specifies the change in rotation (in radians) of the noted nodal degree of freedom.

Returns
Nodal displacements at the current time.
