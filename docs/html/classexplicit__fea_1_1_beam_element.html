Detailed Description

Parent class of Euler-Bernoulli and Timoshenko beam elements.

Each beam element is comprised of 2 node–these refer to indices in a vector of Node's–and a set of properties. Specific to each element is the method of obtaining the stiffness and consistent mass matrices. These methods must be implemented in the respective derived class.

See also
EulerBernoulliBeamElement, TimoshenkoBeamElement`
