unsigned int explicit_fea::ExplicitSystemManager::Options::save_frequency

Frequency at which to save the state of the model, default = 0.

The state of the system will be saved every time i % save_frequency == 0 where i is the current iteration of the system. Set to 0 to disable saving intermediate system states during simulation.

std::string explicit_fea::ExplicitSystemManager::Options::state_filename

Prefix to filename when saving system state, default = "state".

The system state will be saved to the file name state_filename + "_%d.json" at the specified save_frequency as well as the first and last time increment.

bool explicit_fea::ExplicitSystemManager::Options::verbose

Flag to print status updates during simulation, default = false.

If True status updates will be written to the console at the frequency given by save_frequency.
